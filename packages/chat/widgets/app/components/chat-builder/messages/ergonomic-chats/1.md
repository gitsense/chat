# Ergonomic Chats 101

### Welcome!

Most chat interfaces were designed for short exchanges. GitSense Chat was designed for the kind of work that actually takes time: long prompts, complex codebases, branching ideas, and conversations you'll want to return to.

The features below reduce the friction that builds up in longer, more complex conversations.

### Lessons

[table-of-contents-placeholder]

### 1. Auto-Saving Your Drafts

GitSense Chat automatically saves your drafts as you type. If you navigate away, refresh the page, or close the tab, your message will be waiting when you come back.

### 2. Resizing the Chat Input Box

When writing detailed prompts or pasting code snippets, a larger input area makes a real difference.

**Try it now:**

1. Locate the **chat input box** at the bottom of the screen.
2. Find the **grabber icon** (six dots arranged in a 2x3 grid) just above the input box.
3. Click and **drag the grabber upwards** to increase the height of the input box.

### 3. Disabling "Send on Enter"

When writing multi-line prompts or pasting code, you want Enter to create a new line - not send the message.

**Try it now:**

1. Find the **"Send on enter"** toggle above the chat input box.
2. Click it to **disable** sending on Enter. You will now need to click the Send button or use `Ctrl+Enter` / `Cmd+Enter` to send.

### 4. Pinned Messages

Save frequently used prompts, code snippets, or templates and insert them into any conversation with a single click.

**Try it now:**

1. Look at the **options bar** located just above the chat input box.
2. Find and click the **"Pinned"** option (indicated by a pin icon).
3. A modal will appear displaying your saved messages.
4. Click on the **description** of any item to instantly append its content to your current message.

### 5. Navigating Long Conversations

Long threads don't have to be a burden. The Messages Tool lets you filter by role, isolate message ranges, and search across the full conversation - more than a simple Ctrl/Cmd-F can offer.

**Try it now:**

1. Look for the floating bar on the left side of the screen, or the **"Messages Tool"** button at the top of the conversation.
2. Click either to open the **Messages Tool** modal. The number shown in the floating bar is the current message count.
3. In the modal, you can:
   - See a list of all messages
   - Search by keywords
   - Filter by **Role** (User, Assistant, System)
   - Select specific messages using checkboxes

### 6. Building Your Knowledge Tree

GitSense Chat lets you link chats, notes, and tasks into a structured knowledge tree - useful for complex projects where conversations branch into sub-topics or parallel threads.

**Try it now:**

1. Locate the **Quick Chat Buttons** in the right pane.
2. Click **"New Chat"**. A modal will appear.
3. Find the **"Relation"** field and click the dropdown:
   - **Child:** A sub-topic or direct continuation (e.g., diving deeper into a specific bug)
   - **Sibling:** A related topic at the same level (e.g., exploring an alternative solution)
   - **No relation:** A completely independent conversation
4. Click **"Go"** to create the chat and see how it appears in your navigation.
5. Later, click **"Edit Chats"** in the right pane to see a visual tree of all your connections.

### 7. Importing Data Into Your Conversations

Bring files, logs, and code snippets directly into a conversation for easy reference.

**Try it now:**

1. Look for the **"Import New Data"** button in the Quick Chat Buttons (right pane).
2. Click the dropdown arrow next to it and select **"New Message"**.
3. A new message will be added with the Import New Data tool embedded. The tool supports files, logs, and plain text snippets - follow the instructions in the message to complete the import.

### 8. Tracking Code and Context

The **Review Context** button analyzes all messages and shows you exactly which code was generated, which files you added to context, and how your conversation evolved over time.

**Try it now:**

1. If your chat doesn't have any messages yet, send a quick one first - for example: `Write hello world in JavaScript`.
2. Click the **"Review Context"** button in the right pane.
3. The **Chat Context** modal opens with three sections:
   - **Tree Browser (left):** Navigate by **Timeline** or **AI Generated** to find specific code
   - **Metrics (top right):** Totals for messages, tokens, and lines at a glance
   - **Code Table (bottom right):** All generated code blocks with versions and descriptions - use the search bar to quickly find what was added and when

---

These features work together. A well-organized conversation with good context management and reusable prompts isn't just more comfortable - it produces better results.
