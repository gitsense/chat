## Ergonomic Chats 101

### Welcome!

Here you will learn how GitSense Chat was designed from the ground up to make your conversations with LLMs more ergonomic, efficient, and valuable, especially when working with code or complex ideas.

We believe a great conversation interface is key to unlocking the full potential of AI assistance. Let's explore some features that make chatting in GitSense Chat uniquely comfortable and productive.

### Lessons

[table-of-contents-placeholder]

### 1. Auto-Saving Your Drafts

Take your time crafting the perfect message. We automatically save your drafts as you type, so you never have to worry about losing your work if you navigate away or get interrupted.

**Try it now:**

1.  Type a message in the input box below.
2.  Refresh the page.
3.  Notice that your message is still there, ready to send!

### 2. Resizing the Chat Input Box

Crafting detailed prompts or pasting code snippets is much easier with a larger input area.

**Try it now:**

1.  Locate the **chat input box** at the bottom of the screen.
2.  Find the **grabber icon** (six dots arranged in a 2x3 grid) just above the input box.
3.  Click and **drag the grabber upwards** to increase the height of the input box.

### 3. Disabling "Send on Enter"

For longer, multi-line messages, accidentally sending too early can be frustrating.

**Try it now:**

1.  Find the **"Send on enter"** toggle above the chat input box at the bottom of the screen.
2.  Click it to **disable** sending messages with just the Enter key. You will now need to click the 'Send' button or use `Ctrl+Enter`/`Cmd+Enter`.

### 4. Quick Access to Canned Responses & Templates

Stop typing the same prompts repeatedly. The Pinned Messages feature allows you to store frequently used code snippets, standard responses, or complex prompts and insert them into your conversation with a single click.

**Try it now:**

1.  Look at the **options bar** located just above the chat input box.
2.  Find and click the **"Pinned"** option (indicated by a pin icon).
3.  A modal will appear displaying your saved templates and responses.
4.  Click on the **description** of any item to instantly append its content to your current message.

### 5. Taming Long Conversations: Beyond "Ctrl/Cmd-F"

Long threads don't have to be a burden. The Messages Tool offers more than simple text search. Filter by role, isolate message ranges, and gain rapid insights to tame any conversation.

**Try it now:**

1.  Look for the floating bar on the left side of the screen, or click the "Messages Tool" button at the top of the conversation.
2.  Click **"1/1" in the floating bar or the "Messages Tool" button**. The **Messages Tool** modal will open.
3.  In the modal, you can:
    *   See a list of all messages.
    *   Search by keywords.
    *   Filter messages by **Role** (User, Assistant, System).
    *   Select specific messages using checkboxes.
    *   And more!

### 6. Building Your Knowledge Tree

GitSense Chat makes it possible to link chats, notes, and tasks into a structured knowledge tree. Organize complex projects and branching ideas without losing the big picture.

**Try it now:**

1.  Locate the **Quick Chat Buttons** in the right pane of your screen.
2.  Click the **"New Chat"** button. A modal window will appear.
3.  Find the **"Relation"** field in the modal.
4.  Click the dropdown to see your options:
    *   **Child:** Create a sub-topic or a direct continuation (e.g., diving deeper into a specific bug).
    *   **Sibling:** Create a related topic at the same level (e.g., exploring an alternative solution).
    *   **No relation:** Start a completely independent project.
5.  Click **"Go"** to create your new Child chat and see how it appears in your sidebar.
6.  Later, try clicking the **"Edit Chats"** button in the right pane to see a visual tree of all your connections.

### 7. Importing Data Into Your Conversations

GitSense Chat makes it effortless to bring files, logs, and code snippets directly into your conversations for easy reference later.

**Try it now:**

1. Look for the **"Import Data"** button in the Quick Chat Buttons (right pane).
2. Click the dropdown arrow next to it and select **"New Message"**.
3. A new message will be added to your current conversation with the Import Data tool embedded.
4. Follow the instructions provided in the new message to import your files.

### 8. Tracking Code and Context

Understand what's in your conversation at a glance. The **Review Context** button analyzes all messages and shows you exactly which code was generated, which files you added to context, and how your conversation evolved over time.

**Try it now:**

1. First, generate some code to populate your chat. Type: `Write hello world in JavaScript` and send it.
2. Once the assistant responds, click the **"Review Context"** button in the right pane.
3. The **Chat Context** modal opens with three main sections:
   - **Tree Browser (left):** Navigate by **Timeline** or **AI Generated** to find specific code.
   - **Metrics (top right):** See totals for messages, tokens, and lines at a glance.
   - **Code Table (bottom right):** View all generated code blocks with versions and descriptions. Use the search bar to quickly answer: "Did I add this file to context?" or "What code did the AI generate?"
